## Features

- User authentication
- Customer management
- Real-time notifications
- Analytics dashboard

## Technologies Used

- React Native
- Expo
- Redux-toolkit
-Css

## Getting Started

1. Clone the repository:
    ```sh
    git clone https://github.com/Chrismaganga/customer-relations
    ```
2. Install dependencies:
    ```sh
    npm install
    ```
3. Start the application:
    ```sh
    npx expo start -c
    ```
